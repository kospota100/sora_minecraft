
# sora_minecraft

用來存放各種跟Minecraft有關的東西

## 檔案樹 File Tree
```
               File Tree
 =====================================
+ file
    + coordinates                           - 座標的紀錄
        + overworld                         - 主世界
            000_full_scale__全尺寸.xlsx - 全尺寸座標地圖。劃分為 6x6 共 36 塊
            001_center__中央.xlsx - 中央區塊地圖。劃分為 4x4 共 16 塊
            002_NW__西北.xlsx           - x: -3000up <~> -0 // z: 0 <~> 3000up。劃分為 3x3 共九塊
            003_NE__東北.xlsx           - x: 0 <~> 3000up // z: 0 <~> 3000up。劃分同上
            004_SW__西南.xlsx           - x: -3000up <~> -0 // z: -3000up <~> -0。劃分同上
            005_SE__東南.xlsx           - x: 0 <~> 3000up // z: -3000up <~> -0。劃分同上
            010_text__文字資料.csv      - 用文字紀錄的座標與座標內容
            011_text__文字資料.xlsx     - 用文字紀錄的座標與座標內容
        + the_nether                        - 地獄
            [與上面目錄相同]
        + the_end                           - 終界
            [與上面目錄相同]
    todo_list.md                            - 待辦清單
    memo.md                                 - 備忘錄
    + backup                                - 地圖備份
        + saves
```
